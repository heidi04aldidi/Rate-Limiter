1. each userID (our ip address here) is mapped with an object. ie, the userID acts as the key and the object contains the required variables for that user like..
- count = 1
- time = now()

eg: 1072.0.0.1 = {
    count = 1,
    time = now()
}

2. if userID doesn't exist in map, we initialise the userID object with the default initial values like count = 1, startTime = now
